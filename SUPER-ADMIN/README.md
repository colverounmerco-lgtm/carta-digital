# Carta Digital — Guía del Super-Admin

Esta carpeta contiene todo lo que necesitas saber para gestionar la plataforma como propietario.

## Archivos en esta carpeta

| Archivo | Contenido |
|---------|-----------|
| `01-acceso.md` | Cómo entrar al panel, credenciales, cambiar contraseña |
| `02-panel.md` | Qué ves en el panel y qué significa cada dato |
| `03-acciones.md` | Qué puedes hacer con cada restaurante |
| `04-produccion.md` | Configuración en Railway (servidor en la nube) |
| `05-seguridad.md` | Buenas prácticas para proteger la plataforma |
| `06-correo.md` | Credenciales de Gmail para envío de códigos |

## Acceso rápido

| Entorno | URL del panel |
|---------|--------------|
| Local (tu computadora) | http://localhost:5001/admin |
| Producción (Railway) | https://TU-DOMINIO.railway.app/admin |

**Usuario por defecto:** `admin`  
**Contraseña por defecto:** `admin1234` ← cámbiala antes de subir a producción
